# アプリケーション名
Rese（リーズ）
飲食店予約サービス
![top](https://github.com/wa777curry/atte/assets/136479019/c27cd2b9-c054-4484-bb9b-f0866e29c039)

## 作成した目的
外部の飲食店予約サービスは手数料を取られるので自社で予約サービスを持ちたい

## アプリケーションURL
ログインはメールアドレス、パスワードで行う  
パスワードは8文字以上

## 機能一覧
* 会員登録
* ログイン
* ログアウト
* 勤務開始
* 勤務終了
* 休憩開始
* 休憩終了
* 日付別勤怠情報取得
* ページネーション

## 使用技術
Laravel ８．８３．２７
